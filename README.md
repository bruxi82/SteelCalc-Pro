# 🏭 SteelCalc Pro: Universal Business Configurator

**SteelCalc Pro** is a high-performance, lightweight web application designed for construction companies specializing in steel garages, sheds, and modular buildings. 

This tool transforms complex pricing structures into a seamless, 10-second quoting experience.

## 🌟 Why this project?
While many businesses still rely on manual Excel sheets, **SteelCalc Pro** offers a modern, client-facing solution that:
- **Reduces Human Error:** Automated logic based on dimensions and material types.
- **Scalable Architecture:** Easily adaptable for any company by simply adjusting the pricing constants in the code.
- **Client-Ready:** Generates professional PDF offers instantly, optimized for high conversion.

## 🛠️ Key Technical Features
- **Vanilla JS Engine:** Zero dependencies, lightning-fast performance.
- **Responsive Logic:** Handles dynamic surcharges (height, area-based calculations).
- **Print-Optimized CSS:** A dedicated view for professional PDF generation.
- **Security-First:** Built-in trial-period mechanism for software-as-a-service (SaaS) business models.

## 📊 Business Adaptability
This is a **Demonstration Version**. The architecture is built to be customized:
- **Custom Pricing:** All price points can be configured in `script.js`.
- **Branding:** Company-specific logos and contact info can be integrated in minutes.
- **Feature Toggle:** Add or remove options (windows, doors, roofing types) based on inventory.
